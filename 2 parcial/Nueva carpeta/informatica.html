Carrera de Informática

Especialización enfocada en programación, desarrollo web, redes, mantenimiento y tecnologías digitales avanzadas.

¿Qué es Informática?

La carrera de informática prepara estudiantes capaces de crear sistemas, páginas web, programas y soluciones tecnológicas. También se aprende sobre hardware, redes, mantenimiento y diseño digital.

Lo que aplicamos

En la carrera realizamos proyectos de programación en Java, creación de páginas web con HTML y CSS, instalación de redes, mantenimiento de computadoras y diseño gráfico utilizando programas profesionales.

Áreas de Aprendizaje

Programación

Desarrollo de lógica computacional y programas utilizando Java y algoritmos avanzados.

Desarrollo Web

Creación de páginas web modernas utilizando HTML, CSS y diseño responsive.

Redes

Instalación de cableado estructurado, configuración de redes y conectividad.

Mantenimiento

Reparación y mantenimiento de computadoras, software y hardware.

Diseño Digital

Uso de herramientas como Photoshop y CorelDraw para proyectos creativos.

Seguridad

Protección de sistemas, manejo seguro de datos y prevención de amenazas digitales.
